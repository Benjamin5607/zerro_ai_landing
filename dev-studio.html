Desktop App (Windows / macOS / Linux)

Clone the main repo and run the Electron shell — or download a release build when available.

git clone https://github.com/Benjamin5607/zero_ai.git
cd zero_ai
npm install
npm run desktop:dev

Production installer:

npm run desktop:pack:win    # Windows .exe
npm run desktop:pack:mac    # macOS .dmg
npm run desktop:pack:linux  # Linux AppImage

Releases: github.com/Benjamin5607/zero_ai/releases

CLI — advanced / CI

For scripts, headless agents, and power users who prefer the terminal.

macOS / Linux

Windows (PowerShell)

irm https://raw.githubusercontent.com/Benjamin5607/zerro_ai_landing/main/zerro-dev-studio/install.ps1 | iex

Windows (CMD)

curl -fsSL https://raw.githubusercontent.com/Benjamin5607/zerro_ai_landing/main/zerro-dev-studio/install.cmd -o %TEMP%\zerro-install.cmd && %TEMP%\zerro-install.cmd
