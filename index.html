Local Setup

Open Your Local Port

ZerroAI runs in the browser. To allow it to connect to your local Ollama,
you must start your Ollama server with cross-origin enabled.

Run this command in your terminal:

OLLAMA_HOST="0.0.0.0" OLLAMA_ORIGINS="*" ollama serve

Make sure the standard Ollama app is fully closed before running this command.
